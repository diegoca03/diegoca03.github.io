Tipos de selectores que entran en un examen.

  • Clases (class="")

    • Sirve para que aquellas etiquetas, por ejemplo (strong class="texto"), después en estilos pongamos .texto y le demos sus propiedades.

  • Identificador (id="")

    • Cuando ponemos a diferentes etiquetas una misma id, ejemplo (h1 id="estilo_1"), en estilos pongamos estilo_1 y todas las etiquetas que contengan esta id se les aplicarán las propiedades que queramos.

  • Todos los elementos (*)

    • Sirve para seleccionar a todos los h1,por ejemplo.

  • Todos los elementos (p, em, h1, h2)

    • Aquellas etiquetas, tanto p, em, h1 y h2 tengan las misma características.

  • Elementos contenidos en otros elementos (ol li)

    • Aquellas li solamenta de las listas ol apliquen las propiedades que les asignemos.

  • Sobrevuelo (p:hover)

    • En este caso quiere decir que cuando pasamos el ratón por encima del texto de la etiqueta p cambie las propiedades.
